Setup Notes:
-------------


Install node.js: https://nodejs.org/en/download/

Install latest Typescript: npm install -g typescript@latest

	Note: You'll need to delete PATH environment variable pointing to 1.0 typescript

Install ts-node: npm install -g ts-node


Edit with Visual Studio Code: https://code.visualstudio.com/


npm run dev 
npm run prod
